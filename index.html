eventplan by fpaul

generic event planner based on the google appengine

This web application is a planner for events (i.e. parties, festivals, etc.). The person who wants to plan an event, can design all the desired information that is relevant for planning the event. This software creates a form, that confronts all the events attendee for that information. Based on that information, the planning person has the ability to concretely plan the event in detail. Basic features: - This application should be able to communicate with the attendee by sending emails. - It should report some information to the event planning person (i.e. by using the google chart api). - The planning person should be able to design a specific data structure, that fits the type of the event.
